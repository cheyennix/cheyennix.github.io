My website...!

Date: 02-07-2024
Welcome to my website. I'm Cheyenne.

On this website, I plan to post random Thoughts, post about my own Creations, and general stuff like that. Maybe you care because you like taking a look into some other person's life. Maybe you don't care at all. Either is fine. Thank you for being here regardless.

Currently, the site is WIP, but I plan to flesh it out over time with various things.

The theming of the website is provided by deathau. It's a theme for Obsidian, a markdown text editor that's pretty cool. The files of this site exist as markdown files on my hard-drive, but are converted into HTML via a plugin for Obsidian created by KosmosisDesire, which also converts the theme mentioned above into HTML.
